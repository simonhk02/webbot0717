## 🏗️ 技術架構

### 系統架構
```
┌─────────────────┐    ┌─────────────────┐    ┌─────────────────┐
│   前端界面      │    │   API Gateway   │    │   後端服務      │
│  (React/Vue)    │◄──►│   (Express)     │◄──►│  (Node.js)      │
└─────────────────┘    └─────────────────┘    └─────────────────┘
                                │                        │
                                ▼                        ▼
                       ┌─────────────────┐    ┌─────────────────┐
                       │   數據庫層      │    │   外部服務      │
                       │  (PostgreSQL)   │    │  (WhatsApp API) │
                       └─────────────────┘    └─────────────────┘
```

### 核心技術棧
- **前端**：HTML5, CSS3, JavaScript, 響應式設計
- **後端**：Node.js, Express.js, 依賴注入架構
- **數據庫**：SQLite (開發), PostgreSQL (生產)
- **AI服務**：Anthropic Claude API
- **消息服務**：WhatsApp Business API
- **監控**：自建監控系統 + 日誌聚合

### 安全架構
- **認證**：JWT Token + 多因素認證
- **加密**：AES-256 端到端加密
- **審計**：完整的操作日誌記錄
- **合規**：GDPR, SOC2 合規準備
