## 🚀 快速開始

### 第一步：註冊帳戶
1. 訪問 [WhatsApp Bot 官網](http://localhost:3002)
2. 點擊「立即開始」按鈕
3. 填寫基本信息：
   - 電子郵件地址
   - 密碼（至少8位字符）
   - 公司名稱
   - 用戶數量
4. 選擇適合的定價方案
5. 完成註冊

### 第二步：連接 WhatsApp
1. 登入系統後，進入設置頁面
2. 點擊「連接 WhatsApp」
3. 掃描顯示的 QR 碼
4. 確認連接成功

### 第三步：開始記帳
1. 在 WhatsApp 中發送收據圖片
2. 系統自動識別並分類
3. 確認或修改識別結果
4. 完成記帳

## 📱 基本功能

### 1. 收據識別
**功能描述**：自動識別收據內容並提取關鍵信息

**使用方法**：
1. 在 WhatsApp 中發送收據圖片
2. 等待 AI 識別（通常 3-5 秒）
3. 查看識別結果
4. 確認或修改信息

**支援格式**：
- JPG/JPEG
- PNG
- PDF（單頁）

**識別內容**：
- 商家名稱
- 交易金額
- 交易日期
- 商品明細
- 稅號（如有）

### 2. 智能分類
**功能描述**：自動將記帳項目分類到預設類別

**預設類別**：
- 🍽️ 餐飲
- 🚗 交通
- 🛒 購物
- 🏢 辦公
- 🏥 醫療
- 🎓 教育
- 🏠 住宿
- 🎮 娛樂

**自定義分類**：
1. 進入設置 → 記帳類別
2. 點擊「新增類別」
3. 輸入類別名稱和圖標
4. 設置預算限制（可選）

### 3. 數據同步
**功能描述**：多平台數據自動同步

**同步內容**：
- 記帳記錄
- 分類設置
- 用戶偏好
- 報表數據

**同步平台**：
- WhatsApp
- Web 端
- 移動端（即將推出）

## ⚙️ 設置與配置

### 1. 基本設置
**個人信息**：
1. 進入設置 → 個人資料
2. 更新姓名、電話、職位
3. 上傳頭像
4. 設置時區

**通知設置**：
1. 進入設置 → 通知
2. 選擇通知方式：
   - 電子郵件
   - WhatsApp
   - 系統通知
3. 設置通知頻率

### 2. 記帳設置
**自定義問題**：
1. 進入設置 → 記帳設置
2. 點擊「自定義問題」
3. 新增或修改問題：
   - 問題類型（文本、數字、選擇）
   - 問題標籤
   - 是否必填
   - 預設值

**分類規則**：
1. 進入設置 → 分類規則
2. 設置關鍵詞匹配規則
3. 配置自動分類邏輯
4. 測試分類效果

### 3. 安全設置
**密碼管理**：
1. 定期更改密碼
2. 啟用雙因素認證
3. 設置密碼複雜度要求

**會話管理**：
1. 查看活躍會話
2. 強制登出其他設備
3. 設置會話超時時間

### 4. 數據管理
**備份設置**：
1. 設置自動備份頻率
2. 選擇備份內容
3. 配置備份存儲位置

**數據導出**：
1. 選擇導出格式（CSV、Excel、PDF）
2. 設定導出時間範圍
3. 選擇導出字段
4. 執行導出操作

## 🔧 故障排除

### 常見問題解決

#### 1. WhatsApp 連接問題
**症狀**：無法連接 WhatsApp 或連接斷開

**解決步驟**：
1. 檢查網路連接
2. 重新掃描 QR 碼
3. 清除瀏覽器緩存
4. 重新啟動應用程式

**預防措施**：
- 定期檢查連接狀態
- 避免在公共網路使用
- 保持 WhatsApp 版本更新

#### 2. AI 識別失敗
**症狀**：收據識別不準確或失敗

**解決步驟**：
1. 檢查圖片清晰度
2. 確保圖片完整
3. 重新發送圖片
4. 手動輸入信息

**最佳實踐**：
- 使用高解析度圖片
- 確保光線充足
- 避免反光和陰影
- 完整拍攝收據

#### 3. 數據同步問題
**症狀**：多平台數據不同步

**解決步驟**：
1. 檢查網路連接
2. 手動觸發同步
3. 清除本地緩存
4. 重新登入帳戶

#### 4. 性能問題
**症狀**：系統響應緩慢

**解決步驟**：
1. 檢查網路速度
2. 關閉其他應用程式
3. 清除瀏覽器緩存
4. 聯繫技術支援

## ❓ 常見問題

### Q1: 如何更改記帳類別？
**A**: 在記帳記錄中點擊類別名稱，選擇新的類別即可。

### Q2: 可以批量導入記帳數據嗎？
**A**: 是的，支援 CSV 格式批量導入，詳見設置 → 數據管理。

### Q3: 如何設置預算提醒？
**A**: 進入設置 → 預算管理，設置各類別預算和提醒閾值。

### Q4: 支援多種貨幣嗎？
**A**: 目前支援港幣、人民幣、美元，可在設置中選擇。

### Q5: 如何邀請團隊成員？
**A**: 管理員可在用戶管理頁面發送邀請郵件。

### Q6: 數據安全嗎？
**A**: 採用端到端加密，符合 GDPR 標準，定期安全審計。

### Q7: 可以離線使用嗎？
**A**: 目前需要網路連接，離線功能正在開發中。

### Q8: 如何導出報表？
**A**: 在報表頁面選擇時間範圍和格式，點擊導出按鈕。
